# side-scroller-game

### To start the game use server!

### recommended browser - Chrome!

### How to play:
### Your mission is to score the most points. Each enemy tank gives you 10 points. 
### Don't forget to refuel your plane!

### W - up
### S - down
### A - slow speed
### D - fast speed
### SPACE - drops a bomb
### p - pause/play the game

### Enjoy!
